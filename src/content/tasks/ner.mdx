## Core concept

Named Entity Recognition (NER) is a subtask of information extraction that seeks to
locate and classify named entities mentioned in unstructured text into predefined categories
such as person names, organizations, locations.

In the context of historical documents, NER can be particularly challenging
due to the presence of OCR errors and historical language variations.

The Impresso project has developed models and tools to address these challenges and improve
NER performance on historical texts.

## Models

the Impresso project has developed specialized models designed for high-performance extraction from digitized archives. The flagship model is a Stacked BERT-based token classification for named entity recognition, [ner-stacked-bert-multilingual-v1.1.0](https://huggingface.co/impresso-project/ner-stacked-bert-multilingual-v1.1.0) and with support for multilingual historical texts.
This approach allows for robust token classification across **French**, **German**, and **English**, ensuring that both coarse-grained categories (like locations) and fine-grained entities (like specific titles) are captured accurately despite the "noise" of historical datasets.

There is a light version of the model,[Bert Multilingual light](https://huggingface.co/impresso-project/ner-stacked-bert-multilingual-light), but it requires the input already tokenised.

result.df  # View entities as DataFrame
