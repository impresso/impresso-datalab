## TLDR Impresso Py tool

This Python method embeds a text into a vector space and returns a base64 string representing the embedding, prefixed with model tag.

https://impresso.readthedocs.io/en/latest/resources/#impresso.resources.tools.ToolsResource.embed_text

## Core concept

Think of it as a "Semantic Map." Instead of treating words as just letters,
the system assigns the text a location in a high-dimensional space.

In this space, sentences with similar meanings (e.g., "The weather is sunny" and "It is a bright day") are placed physically close to each other, even if they don't share any of the same words.
This enables digital tools to understand "what the text is about" rather than just "how it is spelled."
