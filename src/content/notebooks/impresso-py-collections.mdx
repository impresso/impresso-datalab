## Pagination

{/* cell:4 cell_type:code */}
```python
result = impresso.collections.find(
    limit=5,
)

print(f"Total items in the result set: {result.total}")
for page in result.pages():
    print(
        f"Got page {page.offset} - {page.offset + page.size} of {page.total}. "
        + f"The first title is {page.raw['data'][0]['title']}"
    )
```

{/* cell:8 cell_type:markdown */}
## Add item to collection

{/* cell:9 cell_type:code */}
```python
content_item = impresso.content_items.get("NZZ-1794-08-09-a-i0002")
impresso.collections.add_items(test_collection_id, [content_item.pydantic.id])
```

## Get collection items

Get items from a collection by its ID.

{/* cell:11 cell_type:code */}
```python
items = impresso.collections.items(test_collection_id)
items
```

{/* cell:12 cell_type:markdown */}
## Remove items from collection

{/* cell:13 cell_type:code */}
```python
impresso.collections.remove_items(test_collection_id, [content_item.pydantic.id])
```

## Create, Rename, Delete a collection

{/* cell:17 cell_type:code */}
```python
import uuid

new_collection = impresso.collections.create(
    title=f"Test {uuid.uuid4()}",
    description="This is a test collection.",
)
new_collection
```

{/* cell:18 cell_type:code */}
```python
impresso.collections.get(new_collection.pydantic.id)
```

{/* cell:19 cell_type:code */}
```python
new_collection = impresso.collections.rename(new_collection.pydantic.id, f"Renamed {uuid.uuid4()}")
new_collection
```

{/* cell:20 cell_type:code */}
```python
impresso.collections.delete(new_collection.pydantic.id)
```
